This document describes APIs for the com.samsung.android.sdk.shealth package of Samsung Digital Health SDK.
The packages above contain classes and interfaces which enable you to create applications with trackers on S Health. It provides the followings.

  • Tracker definition and its tile design based on tile templates
  • Posting or removing a tracker tile
  • Handling events related to a tracker tile

For more information related to the development environment, fundamentals, features, and sample description, see Programming Guide.

You can refer to Tracker Design Guidelines to design your tracker tile in the same location.

Those documents above are in the downloaded SamsungDigitalHealthSdk/SHealthService/Docs folder also.
